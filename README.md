# E-Commerce App (REST API)

### General Info:
<p>This back-end project is a music shop e-commerce REST API, that allows users to perform various CRUD operations on products, user accounts, carts, and orders.</p>

### Technologies:
* Node.js
* Express.js
* PostgreSQL

### Technologies (Others):
* Postman
